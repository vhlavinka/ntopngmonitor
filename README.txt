CSC 495 FINAL PROJECT
by Valerie Hlavinka and Patrick Savella

This program utilizes NTOPNG to monitor and store trafic data from an IP camera

To use:
monitor.py : will monitor and collect data and also store the data in a serialized dictionary
minmaxavg.py : will load the serialized dictionary and perform calculations
graphData.py : will load the serialized dictionary and plot the data of throughput over time
detect.py : will gather data and detect whether or not someone is in the scene
"other way" Folder : This folder contains previous versions of each program
